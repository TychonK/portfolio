Portfolio

  • Oh My Donut - Group project

    Oh My Donut is a group project website in which I performed as a team leader. The website has a neat adaptive design. It highlights HTML and CSS (SASS) knowledge through its complex forms, multiple layers background, and adaptivity. Being a team leader, I was challenged to manage my team, establish online meetings, keep track on each teammate's individual progress..., assigning tasks, considering current project condition, to those who have done their part and/or have more spare time. Moreover, creating and handling the Trello desk was under my responsibility. Besides, I reviewed each commit and merged the new code with the old one. I was keen to help every group member with solving any problems they faced. This project is significant, as it made me develop not only crucial technical knowledge in the field of git | GitHub group work management, but also leadership skills, as well as communication and supportiveness.

  • Event Booster - Group project

    Event Booster is another team project. Here, I performed as HTML, CSS (SASS), and JS developer. For this work, emphasis was put on JavaScript and communication with a database through Rest API. This time I not only wrote markup of header and footer but also took part in the development of necessary JS code for API requests. I took care of making the search input work..., along with the country filter. Another unique feature that I implemented was a scroll-up button that appears when user scrolls the window down. Although I was not the team leader in this project, I still helped every teammate with their part of the job. To my delight, I was proclaimed as "Technical King" among my group. This project is substantial as it demonstrates a wide range of JavaScript skills. Furthermore, it prepared me for big, time-consuming works by teaching me patience, problem-solving, and further development of attention to detail and creativity.

  • Web Studio - Individual

    Web Studio is a two-page site representing a local web development business. This was my first individual project which includes all: HTML, CSS (SASS), and JS coding. It shows extensive knowledge in these fields. In particular, HTML and CSS best practices were used for images and icons. Images were optimized and provided with a scrset to ensure good... looking on different screens. SVG sprite with optimized SVG icons was created for numerous vectors storage. All the elements are having custom-written CSS styles. I personally was responsible for handling all the tasks. The distinctive feature of a given project is its ability to store information filled by user in the form inputs. It is done by saving JSON format text in the locale storage. This prevents unwanted deletion of information in case of random internet cut-off. Additionally, the throttling of storing function was implemented to restrict strain on the browser. It was made with the use of the "Lodash" library. The second page with Web Studio's portfolio has working filtering options. Through the creation of this website, I learnt to be self-reliant, improved my patience, and developed research skills.

  • MNTN Landing Page - Individual

    MNTN is a landing page project focusing on attractive design. The main feature of this website is its background parallax scroll effect. I implemented such visual effect with vanilla JavaScript code. This work was all done by myself. Besides its technical aspect, this project gifted me an improvement of research skills together with problem-solving.
